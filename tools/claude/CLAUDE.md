# Global Claude Code Instructions

## Language & Communication

- Communicate in Russian when the user writes in Russian
- Be concise and direct
- Focus on practical solutions

## Code Style

- Write clean, readable code
- Prefer simplicity over complexity
- Use meaningful variable names
- Add comments only when logic isn't self-evident

## Tools & Preferences

- Editor: Neovim
- Shell: Zsh with zinit
- Terminal multiplexer: tmux
- Git UI: lazygit
- Package manager: Nix (cross-platform), Homebrew (macOS)

## Workflow

- Always read files before editing
- Prefer editing existing files over creating new ones
- Run tests after making changes when applicable
- Use git conventional commits (feat, fix, imp, refactor, docs, test)
